🪐 NebulaMart Backend

Backend API and real-time WebSocket server for the NebulaMart artifact auction platform.

Stack

  • Node.js + Express for HTTP serving
  • ws for WebSocket real-time bidding
  • Environment-based configuration (see .env)

Getting Started

npm install
npm start

WebSocket Protocol

Connect to ws://<host>/auction?room=<ROOM_ID> with a valid room ID.

Send JSON messages:

// Place a bid
{ "type": "bid", "amount": 5000 }

// Keep-alive
{ "type": "ping" }

Environment Variables

Copy .env.example to .env before running. See .env for required variables.

About
