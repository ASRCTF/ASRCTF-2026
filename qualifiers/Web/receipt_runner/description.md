Challenge Title: Receipt Runner<br>
Author: Chthonic Cream<br>
Description:<br>
Waystation Orrery routes cargo reimbursements through a quiet supply portal while the main relay array is down. Crew accounts only see their own ration chits, but old command manifests still drift in the same ledger for audit sync.<br>
<br>
The incident board marks this as a manifest-indexing failure. Command receipts were imported with old numeric IDs, and the public audit link allegedly leaks which ID range belongs to command staff. Build a receipt crawler, map the gaps, then request the stale command manifest directly once the pattern stabilizes.<br>
